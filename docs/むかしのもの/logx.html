なんだか久し振りに（はじめて？）FM音源の話。初歩だけど。

FM音源は、メモ帳で値を入力して音色をつくります。
しかし、そのFM音色を作るにあたって、たくさん入力すべき値があってよく分からないですよね。
実は、音源の本質に反映される値は5種類ほどしかありません。

例えば、下のような音源があったとします。
'@ 4
'@ 30, 0, 0, 0, 0, 29,0, 1,0
'@ 30,14, 3,15, 3, 0,0, 1,0
'@ 30,14, 3,15, 3, 0,0, 1,0
'@ 30,14, 3,15, 3, 0,0, 1,0
'@ 5,7

何もFM音源を弄ったことがない人はもうお手上げですよね（´∀｀；
でも、この中の14個の値だけを変えることによってFM音源は構成できるんです。

FM音源の書式はこちら
'@ No
'@ AR,DR,SR,RR,SL,TL,KS,MUL,DT
'@ AR,DR,SR,RR,SL,TL,KS,MUL,DT
'@ AR,DR,SR,RR,SL,TL,KS,MUL,DT
'@ AR,DR,SR,RR,SL,TL,KS,MUL,DT
'@ ALG,FB

これもまた分かりませんよねｗ　しかし、
ここに書いてある「TL」「MUL」「DT」「ALG」「FB」の値を変えるだけでFM音源を弄ることができます。
あとは30でも8でも0でも好きなように値を入力してもいいのです（本当は良くないけど（´∀｀；）。
